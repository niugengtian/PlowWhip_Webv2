### R-010 删除、运维与安全

- **状态：** `REQUIREMENT_OPEN`
- 提供 Goal 和 Task 删除接口，但仅允许满足删除条件的终态对象；删除影响、关联证据和审计记录必须明确。
- Host Bridge 只允许注册项目根目录内的受控操作，路径必须规范化并防逃逸；密钥不得进入日志、报告或诊断包。
- 部署、提交、远端 main、tag/release 和 8742 当前运行镜像分别验收，不能用其中一个代替其他状态。
